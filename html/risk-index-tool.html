FHWA VAST framework

Risk is built from four sub-indices, weighted alongside Criticality (Sensitivity ~10-20%, Criticality ~30%). Adaptive Capacity is reverse-scored (high capacity = low vulnerability).

Master PROTECT Index = f(Exposure, Sensitivity, Asset Value, Adaptive Capacity) weighted with Criticality

Exposure

  • Wildfire pathways (commercial fire modeling)
  • Flood inundation & scour (HEC-RAS)
  • Avalanche / landslide / debris-flow runout (RAMMS)
  • Post-fire erosion & sediment (WEPP)

Sensitivity

  • Bridge / culvert condition, pavement age
  • Equity Study (vulnerable populations, Justice40 multiplier)

Asset Value

  • Strategic / emergency assets, transit
  • Freight, cascading power / telecom

Adaptive Capacity

  • Network redundancy / detours (commercial mobility data)
  • Reverse-scored

Full methodology, data inventory, and the hazard impact & sensitivity matrix live in the reference hub and data & model inventory.

About this tool

The PROTECT Risk Index Tool is the Task 3.3 deliverable of TRPA's PROTECT Plan (FHWA Promoting Resilient Operations for Transformative, Efficient, and Cost-saving Transportation grant). It visualizes the FHWA VAST-based risk index across the Lake Tahoe Region's transportation network for technical partners.

Built by TRPA (Science & Data + Transportation) with ICF as climate-resilience consultant, on the standard TRPA dashboard stack (Calcite + ArcGIS Maps SDK + Plotly + AG Grid). Live spatial data from maps.trpa.org.

Internal review draft. Not for public distribution. Phase 1 MVP for the Steering Committee kickoff; Workshop 1 release November 2026.
